# 基于javaweb学生信息管理系统

#### Description
技术实现：Servlet、JSP、JDBC、DbUtils、EasyUI、jQuery、Ajax
系统功能：学生模块：个人信息管理、同学通讯录、成绩查询、修改密码
教师模块：个人信息管理、教师通讯录、成绩登记、成绩统计、成绩导出、修改密码
系统管理员：基础信息管理，学生信息管理、教师信息管理、考试管理、后台管理
